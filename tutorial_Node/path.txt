1). path.basename(path, extention) - используется для получении части имени файла пути к файлу. Завершающие
                                разделители каталогов игнорируются при использовании этого метода.

2). path.dirname(path) - используется для получении имени каталога по заданному пути.

3). path.extname(path) - используется для получении части расширения пути к файлу. Строка расширения, 
                            возвращенная от последнего появления точки (.) в пути до конца строки пути. 
                            Если в пути к файлу нет точек, возвращается пустая строка.

4). path.format(pathObject) - используется для возврата строки пути из заданного объекта пути.
                             В методе есть несколько правил, согласно которым одно свойство пути 
                             имеет больший приоритет над другим:
                            Параметр «root» объекта пути игнорируется, если указан параметр «dir».
                            Параметры «ext» и «name» объекта пути игнорируются, если указан параметр 
                            «base».

5). path.isAbsolute(path) - используется для того, является ли данный путь абсолютным или нет. Абсолютный
                            путь определяется как путь, содержащий полную информацию, необходимую 
                            для поиска файла.

6). path.join([...paths]) - используется для объединения нескольких сегментов пути с использованием
                             разделителя, зависящего от платформы, для формирования единого пути.

7). path.normalize(path) - используется для нормализации заданного пути.

8). path.parse(path) - использует для возврата объекта, свойства которого представляют заданный путь.
                        Этот метод возвращает следующие свойства:
                        root (root name)
                        dir (directory name)
                        base (filename with extension)
                        ext (only extension)
                        name (only filename)

                        корень (имя корня)
                        директор (имя каталога)
                        база (имя файла с расширением)
                        ext (только расширение)
                        имя (только имя файла)


9). path.relative(from, to) - используется для поиска относительного пути от заданого пути к другому пути
                        на основе текущего рабочего каталога. Если оба заданных пути одинаковых, он 
                        будет преобразован в строку нулевой длинны.

10). path.resolve([...paths]) - используется для преобразования последовательности сегментов пути в 
                        абсолютный путь. Он работает, обрабатывая последовательность путей справа налево, 
                        добавляя каждый из путей перед созданием абсолютного пути. Результирующий путь 
                        нормализуется, а завершающие косые черты удаляются по мере необходимости.

11). path.toNamespacePath(path) - используется для поиска эквивалентного пути с префексом пространства
                        имен из заданного пути. Этот метод имеет смымол только в системах Windows.

12). path.delimeter - используется для получения разделителя пути для конкретной платформы.
