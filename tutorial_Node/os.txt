1). os.EOL - используется для получения символа конца строки или маркера, как указано операционной 
            системой.

2). os.arch() - используется для архитектуры ЦП компьютера, для которого скомпилирован текущий Node.js

3). os.cpus() - используется для получении информации о каждом логическом ядре ЦП компьютера.

4). os.endiannes() - используется для получения порядка байтов ЦП компьютера, для которого скомпелирован 
                    Node.js

5). os.freemem() - используется для получении объема свободной системной памяти.

6). os.getPriority(pid) - используется для получении приоритета планирования процесса, указанного pid. 

7). os.homedir() - используется для получения пути к домашнему каталогу для текущего пользователя.

8). os.hostname() - используется для получении имени хоста операционной системы.

9). os.loadavg() - используется для получения средней нагрузки. Средняя нагрузка это мера активности
                    системы, выраженная дробным числом, вычесляемая операционной системой.

10). os.networkInterfaces() - используется для получения информации о сетевых интерфейсах компьютера.

11). os.platform() - используется для получения платформы операционной системы.

12). os.release() - используется для получения версии операционной системы.

13). os.setPriority(pid, priority) - используется для установки приоретета планирования процесса,
                    указанного pid и приорететом.

14). os.tmpdir() - используется для получения пути к каталогу по умолчанию для временных файлов
                    операционной системы.

15). os.totalmem() - используется для получения общего объема системной памяти в байтах.

16). os.type() - используется для получении имени операционной системы.

17). os.uptime() - используется для получении времени безотказной работы операционной системы в секундах.

18). os.userInfo(options) - используется для получении информации о текущем актианом пользователе.

19). os.version() - используется для определении версии ядра операционной системы.

20). os.constants - используется для получения указанных для операционной системы констант для кода 
                    ошибки, констант сигнала, константы приоритета, константы dlopen и т.д.
